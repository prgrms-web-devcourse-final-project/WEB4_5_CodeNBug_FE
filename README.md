# Code & Bug Frontend

> Code & Bug는 공연·전시·스포츠 등 좌석 예매형 행사의 대규모 동시접속 트래픽을 안정적으로 처리하는 B2B SaaS 티켓팅 플랫폼의 프론트엔드 레포지토리입니다.

---

## 기술 스택

- React v19
- React Router v7
- @tanstack/react-query v5
- Tailwind v4 + Shadcn
